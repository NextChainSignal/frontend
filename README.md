# Frontend

## How to Use

Clone it.

```bash
git clone https://github.com/NextChainSignal/frontend
```
## Prerequisites:
```bash
yarn >= 4.5.0
node >= 22
```

### Install dependencies

```bash
yarn install
```

### Run the development server

```bash
yarn dev
```
